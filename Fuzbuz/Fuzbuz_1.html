Написать функцию dscount, которая подсчитывает количество
идущих подряд символов s1 и s2 в строке, без учёта регистра.
Функция должна пройти следующие тесты, как минимум:

['ab___ab__', 'a', 'b'] => 2

['___cd____', 'c', 'd'] => 1

['de_______', 'd', 'e'] => 1

['12_12__12', '1', '2'] => 3

['_ba______', 'a', 'b'] => 0

['_a__b____', 'a', 'b'] => 0

['-ab-аb-ab', 'a', 'b'] => 2

['aAa', 'a', 'a'] => 2
